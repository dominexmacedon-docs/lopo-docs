## [1.0.0] - 2026-07-01
### Added
- Initial release of **Lopo Editor v1.0.0**
- Installer: `Lopo-v1.0.0.exe`
- Core language interpreter with variables, expressions, conditionals, loops, functions
- Support for entities (object‑like structures), arrays, and objects
- Built‑in utility functions
- Command‑line interface
- Native Windows editor with syntax highlighting
- Documentation: `README.md`, `lopo-guide.pdf`
