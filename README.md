# Lopo Programming Language

Lopo is a lightweight, interpreted programming language designed for simplicity, readability, and fast learning.

It allows developers to write clean and minimal code while still supporting powerful programming features such as variables, loops, functions, and entity-based design.

---

##  Hello World Example

```lopo
show("Hello World!");
````

##  Why Lopo?

* Simple syntax
* Easy to learn
* Fast execution (AST-based interpreter)
* Supports modern programming structures
* Built-in CLI tools

---

##  Basic Idea

Lopo focuses on readability:

```lopo
define name = "Lopo";
show(name);
```

---

##  Running Lopo

After downloading:

1. Put `lopo.exe` into a folder (example: `C:\Lopo`)
2. Add it to system PATH
3. Run in terminal:

```bash
lopo hello.lopo
```
