# Lopo Programming Language

Lopo is a lightweight interpreted programming language designed to be simple, readable, and easy to learn. It provides a clean syntax while supporting practical programming features such as variables, functions, loops, entities, arrays, objects, and built-in utilities.

---

## Hello World

```lopo
show("Hello World!");
```

## Why Lopo?

* Simple and readable syntax
* Easy for beginners to learn
* Lightweight interpreter
* Variables, functions, loops, and entities
* Arrays and objects
* Built-in utility functions
* Cross-platform design
* Command-line interface

---

## Basic Example

```lopo
define language = "Lopo";

show("Welcome to " + language + "!");
```

---

## Running a Program

Create a file named `hello.lopo`:

```lopo
show("Hello from Lopo!");
```

Run it from Command Prompt or Terminal:

```bash
lopo hello.lopo
```

You can also double-click a `.lopo` file after installing Lopo on Windows.
